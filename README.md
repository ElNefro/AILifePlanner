# AI Life Planner

AI Life Planner — это AI-платформа для постановки целей, планирования и достижения результатов.

## Проект объединяет:
- интерактивную доску целей;
- систему задач и подзадач;
- AI-анализ целей;
- roadmap-планирование;
- AI-рекомендации;
- напоминания и расписание;
- синхронизацию между устройствами.

---

# Основная идея

Пользователь пишет цель обычным текстом.

Например:

```text
Хочу открыть свой бизнес
```

## AI:
- анализирует цель;
- задает уточняющие вопросы;
- строит roadmap;
- создает этапы;
- генерирует задачи и подзадачи;
- помогает двигаться к результату.

---

# Основные возможности

## Интерактивная AI-доска
